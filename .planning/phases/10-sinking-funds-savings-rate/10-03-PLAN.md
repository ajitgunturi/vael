   - `monthlyMetricsDaoProvider`: Provider returning MonthlyMetricsDao from databaseProvider
   - `savingsRateMetricsProvider`: FutureProvider.family taking (String familyId) that:
     a. Gets last 12 months as YYYY-MM strings (current month and 11 prior)
     b. Fetches cached metrics from MonthlyMetricsDao.getRecent(familyId, 12)
     c. For each target month: if cached entry exists and is not current month and computedAt is after latest transaction updatedAt, use cache. Otherwise compute fresh:

        - Compute savingsRateBp via SinkingFundEngine.savingsRateBp(incomePaise, expensesPaise)
        - Upsert into MonthlyMetrics with id=`${familyId}_${month}`, computedAt=DateTime.now()
     d. Current month is ALWAYS recomputed (never use cache)
     e. Return List<MonthlyMetric> sorted by month ascending
   - `currentMonthRateProvider`: Provider.family that extracts current month from savingsRateMetricsProvider, returns double percentage (savingsRateBp / 100.0)
   - Use TransactionDao for date-range queries (already exists: `getByDateRange`)

      Logic:
      1. Compute baseline FI number via FiCalculator.computeFiNumber
      2. Compute baselineYears via FiCalculator.yearsToFi
      3. downPayment = (purchaseAmountPaise * downPaymentBp / 10000).round()
      4. afterPortfolio = max(0, currentPortfolioPaise - downPayment)
      5. If loan: loanPrincipal = purchaseAmountPaise - downPayment, emi = FinancialMath.pmt(rate: loanInterestRate!/12, nper: loanTenureMonths!, pv: -loanPrincipal).abs()
      6. afterMonthlySavings = max(0, monthlySavingsPaise - emi)
      7. afterYears = FiCalculator.yearsToFi with modified inputs
      8. fiDelayYears = afterYears == -1 ? -1 : afterYears - baselineYears
    - `static int educationCostAtYear({required int baseCostPaise, required int escalationRateBp, required int years})`
      Logic: compound escalation using (baseCostPaise * pow(1 + escalationRateBp/10000, years)).round()

   - Watches `dashboardDataProvider(familyId)` for net worth and savings rate
   - Watches `fiDefaultInputsProvider((userId: userId, familyId: familyId))` for FI inputs
   - Watches `milestoneListProvider((userId: userId, familyId: familyId))` for milestones
   - Watches `efStateProvider((userId: userId, familyId: familyId))` for EF coverage
   - Computes FI number via FiCalculator.computeFiNumber if hasLifeProfile is true
   - Computes yearsToFi via FiCalculator.yearsToFi if hasLifeProfile is true
   - Returns PlanningHealthData with all available metrics, null for unconfigured ones

Each metric is null when its source feature is not configured. The screen uses null to show "Set up" CTAs.
